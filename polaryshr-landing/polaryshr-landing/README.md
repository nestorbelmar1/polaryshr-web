# Polaryshr Landing

## Cómo usar
1. Descomprime el ZIP.
2. Abre terminal en esta carpeta.
3. Instala dependencias:

```bash
npm install
```

4. Ejecuta en local:

```bash
npm run dev
```

5. Sube la carpeta a GitHub y luego impórtala en Vercel.

## Dominio
En Vercel agrega:
- polaryshr.cl
- www.polaryshr.cl

Luego configura en NIC Chile los DNS que te indique Vercel.
